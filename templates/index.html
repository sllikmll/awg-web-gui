AmneziaWG Web GUI

Минимальное управление серверами и клиентами AWG 1.5 / 2.0

Серверы

Версия AWG, контейнер, interface, config path, UDP port, subnet и DNS определяются автоматически из remote config. Удаление сервера удаляет его только из базы панели, реальные контейнеры не трогает.

Клиенты

При добавлении делается backup remote config, запись peer в config и restart только AWG-контейнера.

Мониторинг и трафик

Online, last handshake, endpoint, RX/TX и накопительный трафик по клиентам.
